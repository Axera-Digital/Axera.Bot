# [ Axera.Bot ]
Axera.Bot is a fully-featured Discord bot written in Python. It is designed to be the ONLY bot needed in a server to manange everything from moderation to entertainment and beyond.

## Features
* Server Moderation
* Music
* Games
* Memes
* Server Utilities
* Much More!
